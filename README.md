# 🚀 Terraform Projects

A collection of **Terraform projects on AWS** demonstrating **Infrastructure as Code (IaC)** through hands-on implementations. This repository is organized as a structured learning journey, progressing from foundational concepts to production-ready cloud infrastructure.

This repository serves as my Terraform learning and project portfolio. It contains practical implementations that showcase how to provision, manage, and automate AWS infrastructure using Terraform.

As I continue learning and building, new projects will be added to demonstrate different cloud services, Terraform concepts, and Infrastructure as Code best practices.

## 🚀 Standard Terraform Workflow

Each project will typically follow the Terraform workflow:

```bash
terraform init
terraform validate
terraform fmt
terraform plan
terraform apply
terraform output
terraform destroy
```
